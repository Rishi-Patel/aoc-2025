# Advent of Code 2025

To run, use the run.sh script
```
Usage: ./run.sh <day> [build_type]
Example: ./run.sh day01
Example: ./run.sh day01 release
Example: ./run.sh day01 debug
```

Note: This does not contain input, please download your own input from [AdventOfCode](https://adventofcode.com/).
Store this input in the corresponding day folder. E.g. Input for day01 should be stored as 01/input.txt.
